# CTF Scripts

A collection of scripts for CTF competitions, organised by category.

## Categories

| Category | Description |
|---|---|
| [forensics/](forensics/) | File analysis, steganography, PCAP, memory and disk forensics |
| [cryptography/](cryptography/) | Classical ciphers, RSA attacks, AES attacks, hash attacks |
| [web/](web/) | Web exploitation: SQLi, LFI, SSRF, SSTI, XXE, JWT, and more |
| [binary_exploitation/](binary_exploitation/) | Buffer overflows, ROP, heap exploitation, reverse engineering |
| [osint/](osint/) | Username lookup, DNS, certificate transparency, WHOIS |
| [networking/](networking/) | Port scanning, banner grabbing, service brute-forcing, SMB |
| [misc/](misc/) | Encoding, entropy, flag grepping, wordlists, workflow utilities |
